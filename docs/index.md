# Estudos sobre Fastapi

O objetivo inicial deste projeto é estudar conceitos sobre o Fastapi e documentar novos conceitos, ferramentas e comandos que ainda não estou failiarizado.

Atualmente sou desenvolvedor backend Python, e a intenção deste projeto e aumentar meus conhecimentos.

## Curso de Fastapi com Dunossauro

O primeiro passo dos estudos, decidi acompanhar a série do Dunossauro sobre Fastapi e replicar o código com algumas alterações a meu gosto.
